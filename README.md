# Food-Ordering-Website
A full-stack Food Ordering Web Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This project allows users to browse food items, add items to a cart, place orders, and track them in real time. It also includes an admin panel for managing menu items and orders.
